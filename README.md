📄 Chat with PDF – Frontend

An intelligent web application that allows users to upload PDF documents and interact with them using AI-powered chat. This frontend provides a smooth and responsive interface for document upload, search, and conversational querying.

🚀 Features
📤 Upload PDF files 
💬 Chat with your PDF content using AI
🔍 Search through uploaded documents
🕘 View chat history
⚡ Fast and responsive UI
🌐 Deployed on cloud for real-time access

🛠️ Tech Stack
Frontend: React.js
Backend (API): Node.js, Express.js
Database: Firebase (Firestore)
AI Integration: External API (Gemini / OpenAI)
Deployment: Vercel (Frontend), Render (Backend)


📂 Project Structure
chat-with-pdf-frontend/
│── public/
│── src/
│   ├── components/
│   ├── pages/
│   ├── services/      # API calls
│   ├── App.js
│   └── index.js
│── package.json
│── README.md
